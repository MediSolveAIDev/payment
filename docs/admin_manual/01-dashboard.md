# 01. 대시보드 (Dashboard)

대시보드는 시스템 로그인 후 가장 먼저 마주하는 화면으로, 결제 시스템의 **전체 운영 현황**을 실시간으로 파악할 수 있는 관제 센터 역할을 합니다.

---

## 1. 핵심 지표 (이번 달 현황)

화면 상단에는 이번 달(1일~현재)의 누적 데이터가 4개의 카드로 표시됩니다.

*   **이번 달 총 매출**: 이번 달에 발생한 모든 결제 성공 금액(정기+단건)의 합계입니다.
*   **신규 구독**: 이번 달에 새롭게 시작된 구독 계약의 총 건수입니다. (무료 체험 포함)
*   **결제 실패**: 이번 달에 결제가 시도되었으나 실패한 건수입니다. 높은 수치가 지속될 경우 전체 장애나 특정 카드사 이슈를 의심해볼 수 있습니다.
*   **환불/취소**: 이번 달에 관리자가 직접 처리하거나 시스템에 의해 환불된 금액의 합계입니다.

---

## 2. 실시간 모니터링 레일 (우측 사이드바)

화면 우측의 세로 바는 현재 시스템에서 일어나고 있는 **가장 최근의 개별 이벤트**를 보여줍니다.

*   **최근 구독**: 방금 구독을 시작한 사용자의 ID와 상태(TRIAL/ACTIVE)를 보여줍니다. 클릭 시 해당 구독의 상세 화면으로 이동합니다.
*   **최근 결제**: 발생한 결제 건들을 금액과 함께 보여줍니다. 성공(DONE)은 초록색, 실패(FAILED)는 빨간색 배지로 표시되어 즉각적인 식별이 가능합니다.
*   **미수 구독**: 현재 결제가 실패하여 '미수(PAST_DUE)' 상태에 머물러 있는 사용자들입니다. 이들은 시스템에 의해 자동으로 재결제가 시도됩니다.
*   **만료 임박**: 향후 7일 이내에 구독 기간이 끝날 예정인 사용자들입니다.

---

## 3. 심층 분석 차트

시스템의 중장기 추세를 분석하기 위한 시각화 자료를 제공합니다.

*   **최근 12개월 구독**: 지난 1년간의 전체 구독자 수와 매달 새로 가입한 신규 구독자 수의 비중을 막대 그래프로 비교합니다.
*   **최근 12개월 일반매출**: 구독 외에 단건(일회성)으로 발생한 매출의 변화 추이를 보여줍니다.
*   **구독 상태 비율 (도넛 차트)**: 현재 운영 중인 모든 구독의 상태별(정상 이용 중, 체험 중, 미수 등) 비중을 보여줍니다.
*   **최근 30일 구독 추이**: 지난 30일간 일별로 몇 명이 새로 가입하고, 몇 명이 해지하거나 만료되었는지 라인 그래프로 상세히 보여줍니다.

---

## 4. 서비스별 현황 테이블 (시스템 관리자 전용)

여러 서비스를 통합 관리하는 경우, 각 서비스(상점)별로 이번 달 매출액과 구독자 수를 한눈에 비교할 수 있는 표가 하단에 위치합니다. 특정 서비스명을 클릭하면 해당 서비스의 더 자세한 관리 화면으로 연결됩니다.
