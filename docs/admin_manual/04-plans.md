# 04. 요금제 관리 (Plans)

서비스에서 판매할 구독 상품의 가격, 할인, 체험 정책 등을 설계하고 관리하는 메뉴입니다.

---

## 1. 요금제 목록 및 상태

현재 운영 중이거나 과거에 사용했던 모든 요금제 목록을 확인합니다.

### 테이블 항목(컬럼) 상세 설명
*   **이름**: 요금제명입니다. 클릭 시 수정 화면으로 이동합니다.
*   **서비스**: 해당 요금제를 사용하는 서비스명입니다.
*   **가격**: 정가 기준 금액입니다.
*   **주기**: 결제가 반복되는 기간입니다.
*   **상태**: 현재 해당 요금제의 활성화 여부입니다.

---

## 2. 요금제 생성 및 수정 필드 상세

신규 요금제를 등록하거나 수정할 때 설정하는 항목들에 대한 상세 설명입니다.

### 기본 정보
*   **요금제 이름**: 사용자 결제창과 관리자 화면에 표시될 상품명입니다.
*   **정가 (원)**: 할인이 적용되기 전 기준 금액입니다.

### 결제 주기 (반복회차)
*   **월/년/주 단위**: 해당 기간마다 자동으로 결제가 발생합니다.
*   **일(Day) 단위**: 원하는 일수(예: 30일, 90일)를 직접 지정하여 주기를 설정할 수 있습니다.
*   **주의**: **결제 주기는 요금제 생성 시에만 설정 가능**하며, 이미 생성된 요금제의 주기는 수정할 수 없습니다. 주기를 바꾸려면 새 요금제를 만들어야 합니다.

### 할인 정책
*   **첫구독 할인**: 사용자가 이 요금제를 처음 신청할 때만 적용되는 혜택입니다.
    *   `무료`: 첫 회차는 0원으로 처리됩니다. (체험과는 다름)
    *   `할인 금액/할인율`: 정가에서 일정 금액이나 비율만큼 깎아줍니다.
*   **상시 할인**: 구독 기간 내내 모든 회차 결제에 적용되는 할인입니다.

### 체험 및 갱신 정책
*   **체험(Trial) 제공**: 결제 정보를 등록하면 즉시 결제되지 않고 지정된 일수만큼 서비스를 무료로 이용하게 합니다. 체험이 끝나면 첫 주기가 자동 결제됩니다.
*   **자동결제 안함**: 이 옵션을 켜면 첫 주기가 끝난 후 다음 결제를 시도하지 않고 구독이 자동으로 종료됩니다.

---

## 3. 추가 정보 (Extra Info)

요금제에 부가적인 속성을 관리할 수 있는 기능입니다. (예: 용량: 10GB, 사용자 수: 5명 등)
*   키와 값을 한 줄씩 입력하며, 서비스 화면에서 요금제의 특장점을 설명하는 용도로 활용됩니다.

---

## 4. 금액 미리보기 (Amount Preview)

설정값을 입력하면 하단에 **첫 결제 시 고객이 지불할 금액**과 **다음 회차부터 청구될 정기 금액**이 자동으로 계산되어 표시됩니다. 저장 전 최종적으로 의도한 금액이 맞는지 반드시 확인하세요.
