# Reset

검색바 우측. 다른 위치에도 사용 가능.

---

## 상태별 스타일

| 상태 | bg | border | icon |
|------|-----|--------|------|
| Default | `white` | `gray-300(#E3E3E3)` | `gray-600(#9F9F9F)` |
| Hover | `color-primary-100` | `color-primary` | `color-primary` |
| Active | `white` | `gray-300(#E3E3E3)` | `color-primary` |
