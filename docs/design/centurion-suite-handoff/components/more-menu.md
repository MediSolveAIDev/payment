# More Menu

---

## Variant

| Variant | 메뉴 항목 |
|---------|----------|
| Menu | 수정 + 삭제 (2개) |
| Menu_단일 | 수정만 (1개) |

---

## 상태별 스타일

| 항목 | 상태 | bg | text |
|------|------|-----|------|
| 수정 | Default | `white` | `black` |
| 수정 | Hover | `gray-100(#FBFBFB)` | `black` |
| 삭제 | Default | `white` | `color-red(#FF4E51)` |
| 삭제 | Hover | `gray-100(#FBFBFB)` | `color-red(#FF4E51)` |

---

## 드롭다운 스타일
- Drop Shadow: X `0`, Y `2`, Blur `10`, `#CFCFCF` 50%
