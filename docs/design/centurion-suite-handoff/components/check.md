# Checkbox

---

## 상태별 스타일

| 상태 | bg | border | icon |
|------|-----|--------|------|
| Default | `white` | `gray-300(#E3E3E3)` | — |
| Active | `color-primary` | — | `white` (✓) |
| Disabled | `gray-200(#F3F3F3)` | `gray-300(#E3E3E3)` | — |

---

## 인터랙션
- Active ↔ Default 토글
- Disabled: 상태 전환 불가
