# Process Icon (상태 아이콘)

---

## 타입별 스타일

| 타입 | 형태 | bg | icon | 의미 |
|------|------|-----|------|------|
| Normal | 원형 | `color-primary` | `white` (✓) | 완료, 성공 |
| Critical | 원형 | `color-red(#FF4E51)` | `white` (!) | 오류, 실패 |
| Warning | 삼각형 | `#FFA200` | `white` (!) | 주의, 경고 |

---

## 사용처
- Toast 컴포넌트의 상태 아이콘
- Confirmation Modal의 상태 아이콘
- 테이블 내 상태 표시
