# 기관 코드

토스페이먼츠에서 지원하는 카드사, 은행, 간편결제사, 통신사 코드입니다. 응답은 두 자리 코드만 지원합니다. 요청은 숫자·한글·영문 코드를 지원하지만 간편결제사 코드처럼 일부만 지원하는 케이스도 있습니다. 코드가 있다면 두 자리 코드 사용을 가장 권장합니다.

금융결제원 공식 코드는 세 자리 숫자 코드입니다. 두 자리 코드, 한글 코드, 영문 코드는 토스페이먼츠에서 정의한 값입니다.

전체 기관 코드 다운로드
