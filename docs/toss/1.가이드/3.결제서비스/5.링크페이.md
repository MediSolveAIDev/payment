# 링크페이 이해하기

> **참고**: 링크페이는 추가 계약 후 사용할 수 있습니다. 도입 문의는 아래 버튼을 눌러주세요.도입 문의하기

링크페이는 **개발 없이(no-code)** 결제 링크를 생성해 고객에게 바로 전달할 수 있는 서비스예요.
간단한 정보만 입력하면 결제 링크가 생성되고, 문자나 SNS로 공유해 고객이 바로 결제할 수 있어요.
웹사이트나 앱이 없어도 되니 SNS 판매자, 프리랜서 등 다양한 비즈니스에서 손쉽게 활용할 수 있어요.

필요하면 API를 활용해 기존 시스템이나 비즈니스 로직에 연동할 수 있어요.
간단한 결제부터 확장된 결제 흐름까지 유연하게 구현할 수 있어요.

결제 링크가 실제로 어떻게 보이는지 샘플로 먼저 확인해보세요.

샘플 URL 바로가기

![링크페이 FLOW](https://static.tosspayments.com/docs/glossary/linkpay_flow.png)

title: 링크페이는 어떤 결제수단을 지원하나요?

카드, 계좌이체, 간편결제, 가상계좌 등 다양한 결제수단을 지원해요. 구매자가 결제 링크를 열면 원하는 결제수단을 선택해서 결제할 수 있어요.

title: 결제 링크는 몇 번까지 사용할 수 있나요?

링크페이 결제 링크는 사용 횟수에 제한이 없어요. 다만, 재고 기능을 활용하면 결제 가능 횟수를 제한할 수 있습니다.

title: 링크페이 수수료는 어떻게 되나요?

결제수단별 기본 수수료가 적용돼요. 자세한 수수료는 토스페이먼츠 고객센터(1544-7772, support@tosspayments.com)로 문의해주세요.

title: 문자 발송 기능은 별도 과금되나요?

아니요. 링크페이는 결제 수수료 외 추가 과금되는 금액은 없습니다.

## 링크페이 시작하기

링크페이는 두 가지 방법으로 사용할 수 있어요.

### 상점관리자로 시작하기

개발 없이 상점관리자에서 바로 결제 링크를 생성하고 관리할 수 있어요.

![상점관리자로 시작하기](https://static.tosspayments.com/docs/glossary/linkpay_main.png)

#### 결제 링크 생성하기

1. 상점관리자에 로그인하세요.
2. 링크페이-판매상품 메뉴로 이동하세요.
3. **상품 올리기** 버튼을 클릭하세요.
4. 상품명, 금액 등 필요한 정보를 입력하세요.
5. 생성된 링크를 복사해서 구매자에게 공유하세요. 문자 발송 기능도 제공합니다.

#### 결제 내역 확인하기

상점관리자에서 링크페이로 발생한 결제 내역을 확인할 수 있어요. 결제 상태, 결제 금액, 결제 일시 등을 한눈에 볼 수 있어요.

---

### API로 시작하기

API를 사용하면 결제 링크 생성을 자동화할 수 있어요. 주문 시스템, CRM, ERP 등과 연동해서 결제 링크를 자동으로 생성하고 발송할 수 있어요.

![링크페이 API 연동 플로우](https://static.tosspayments.com/docs/glossary/10_linkflow.png)

#### 연동 흐름

1. **상품(결제 링크) 생성**: 상품 생성 API를 호출하면 `ProductKey`와 결제 URL이 반환돼요.
2. **상품 이미지 업로드(선택)**: 필요하면 상품에 노출할 이미지를 업로드할 수 있어요.
3. **결제 링크 전달**: 반환된 결제 URL을 문자, SNS, 이메일 등 원하는 방식으로 구매자에게 공유하세요. SMS 발송 API를 사용하면 토스페이먼츠에서 대신 문자를 보내드려요.
4. **웹훅으로 결제 결과 수신**: 구매자가 결제를 완료하면 웹훅으로 결제 상태를 알려드려요.

## 활용 사례

| 비즈니스 유형     | 활용 방법                                          |
| ----------------- | -------------------------------------------------- |
| **SNS 판매자**    | 인스타그램, 블로그에서 상품 판매 시 결제 링크 공유 |
| **오프라인 매장** | 비대면 주문, 예약금 결제                           |
| **프리랜서**      | 프로젝트 비용 청구, 인보이스 발송                  |
| **학원/교육**     | 수강료, 교재비 수금                                |
