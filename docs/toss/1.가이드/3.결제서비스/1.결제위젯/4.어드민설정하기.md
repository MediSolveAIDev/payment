## 결제 UI 설정

결제위젯은 [국내 일반결제](/resources/glossary/normal-payment), [브랜드페이](/guides/v2/brandpay), [해외 간편결제](/guides/v2/payment-widget/integration-paypal) 서비스를 지원해요.
각 결제서비스에는 상점아이디(MID)란 고유 아이디가 부여돼요. 결제서비스와 MID는 1:1 관계로 생성돼요. 예를 들어, 일반결제와 브랜드페이 결제서비스를 계약했다면 총 2개의 MID가 발급돼요.

하나의 결제 UI에는 다양한 MID의 [결제서비스를 추가](#결제-ui에-서비스-추가하기)할 수 있어요. 예를 들어, 하나의 결제 UI에 일반결제, 브랜드페이 결제서비스를 둘 다 노출시키는 방법이에요.
각 결제 UI에 하나의 MID를 1:1로 연결시킬 수도 있어요. 새로운 [결제 UI를 만들고](#새로운-결제-ui-추가하기) 연동할 때 해당 UI의 `variantKey`를 파라미터로 설정하세요.

### 결제 UI에 서비스 추가하기

하나의 결제 UI에 2개 이상의 결제서비스를 추가할 수 있어요.

1. 결제 UI 설정 메뉴에서 이용서비스를 추가할 UI 환경에 **이용서비스 > 추가하기**를 선택하세요.&#x20;
   ![결제 UI에 서비스 추가하기 1](https://static.tosspayments.com/docs/payment-widget/admin/add-service-1.png)
2. 팝업창에서 사용하고 싶은 서비스를 활성화한 다음에 해당 서비스용 상점아이디(MID)를 선택하고 **저장하기**를 누르세요.
   ![결제 UI에 서비스 추가하기 2](https://static.tosspayments.com/docs/payment-widget/admin/add-service-2.png)
3. 결제 UI에 서비스가 추가된 것을 확인할 수 있어요. **이용서비스** 위에 마우스를 올리면 연결된 상점아이디(MID)가 보여요.
   ![결제 UI에 서비스 추가하기 3](https://static.tosspayments.com/docs/payment-widget/admin/add-service-3.png)
4. 개발자센터의 [API 키 메뉴](https://developers.tosspayments.com/my/api-keys)에서 확인한 **결제위젯 연동 키**로 [연동을 시작](/guides/v2/payment-widget/integration)하세요.
   ![결제 UI에 서비스 추가하기 3](https://static.tosspayments.com/docs/using-api/widget-key.png)

### 새로운 결제 UI 추가하기

결제 UI를 여러 개 만들고 상황에 따라 다르게 노출하고 싶을 때는 새로운 결제 UI를 추가하세요. MID 별로 다른 결제 UI를 만들거나, 계정이나 상품별로 다른 결제수단을 보여주는 등 조건에 따라 구매자에게 다른 UI를 보여줄 수 있어요.

1. 결제 UI 설정 메뉴에서 UI 추가하기 버튼을 누르세요.
   ![새로운 결제 UI 추가하기 1](https://static.tosspayments.com/docs/payment-widget/admin/add-ui-1.png)
2. 새로운 UI에 추가하고 싶은 결제 서비스를 모두 선택하고 **다음**을 누르세요.
   ![새로운 결제 UI 추가하기 2](https://static.tosspayments.com/docs/payment-widget/admin/add-ui-2.png)
3. 선택한 서비스에 계약된 상점아이디(MID)가 2개 이상이라면, 해당 결제 UI에 사용할 MID를 선택하는 화면이 나와요.
   ![새로운 결제 UI 추가하기 3](https://static.tosspayments.com/docs/payment-widget/admin/add-ui-3.png)
4. 새로운 UI에 사용할 베리언트 키(`variantKey`) 값을 입력하고 **설정하기**를 누르면 결제 UI가 추가돼요.
   ![새로운 결제 UI 추가하기 4](https://static.tosspayments.com/docs/payment-widget/admin/add-ui-4.png)
5. 연동하고 싶은 결제 UI의 `variantKey`를 복사하세요.
   ![새로운 결제 UI 추가하기 5](https://static.tosspayments.com/docs/payment-widget/admin/add-ui-6.png)
6. **결제위젯 연동 키**로 [연동을 시작](/guides/v2/payment-widget/integration)하세요. [`renderPaymentMethods()`](/sdk/v2/js#widgetsrenderpaymentmethods) 메서드에 `variantKey` 파라미터를 설정하세요.
   ```plain copyable="false" theme="grey"
   paymentWidget.renderPaymentMethods(
     "#payment-method",
     { value: 10000 },
     { variantKey: "DEFAULT" }     // 기본 UI
     // { variantKey: "DEFAULT-2" } // 추가 UI
   )
   ```

### UI 세부 설정

각 결제 UI를 다르게 커스터마이징할 수 있어요. 결제위젯 어드민의 디자인 및 기능 메뉴에 있는 설정을 소개합니다.

#### 디자인

결제수단 레이아웃
카드사 레이아웃
기본 설정
색상

결제수단을 3열 또는 2열로 보여주세요. 최상단 강조는 **기능 > 결제수단 목록**의 첫 번째 결제수단을 강조해요.

![토스페이먼츠 결제수단 레이아웃](https://static.tosspayments.com/docs/payment-widget/admin/design-payment.png)

구매자가 **신용・체크카드**를 선택했을 때 카드사 목록을 보여주는 방법입니다. 버튼을 선택하면 각 카드사의 로고와 이름이 버튼으로 보여요. 드롭다운을 선택하면 메뉴에서 카드사 이름이 보여요.

![토스페이먼츠 카드사 레이아웃](https://static.tosspayments.com/docs/payment-widget/admin/design-card.png)

결제위젯 전체 패딩, 제목을 수정할 수 있는 디자인 기본 설정입니다. 기본으로 있는 '결제 방법' 제목도 제거할 수 있어요.

![토스페이먼츠 기본 설정](https://static.tosspayments.com/docs/payment-widget/admin/design-basic.png)

프로모션 배지, 체크박스, 브랜드페이(라디오 버튼, 결제수단 설정 페이지)의 강조색과 대조색, 테마를 변경할 수 있어요.

![토스페이먼츠 디자인 색상](https://static.tosspayments.com/docs/payment-widget/admin/design-colors.png)

카드사 목록

Pro 기능입니다.

![토스페이먼츠 결제위젯 결제수단 설정](https://static.tosspayments.com/docs/payment-widget/admin/func-payment.png)

노출하고 싶은 카드사를 선택하세요. 할부 고급설정에서 각 카드사의 최대 할부기간, 무이자 할부 기간을 설정할 수 있어요.

![토스페이먼츠 결제위젯 카드사 목록](https://static.tosspayments.com/docs/payment-widget/admin/func-cards.png)

가상계좌 환불 정보는 결제창을 띄운 시점부터 30분 동안만 조회가 가능합니다. 자세한 내용은 [결제 취소 가이드](/guides/v2/cancel-payment#case-2-결제위젯-환불-계좌-확인하기)에서 확인하세요.

![결제위젯 가상계좌 설정](https://static.tosspayments.com/docs/payment-widget/integration/virtual-account-setting.png)

결제수단에 프로모션 배지를 추가하고, 프로모션 텍스트로 혜택을 안내하세요. 프로모션 텍스트는 결제위젯 [Pro 기능](/guides/v2/payment-widget/pro)입니다.

## 약관 설정

결제 이용약관을 여러 개 만들고 상황에 따라 다르게 노출할 수 있어요. 예를 들어, [해외결제](/guides/v2/payment-widget/integration-paypal)를 사용하고 있다면 구매자에게 영문 이용약관을 보여주세요.

**상점관리자 > 결제위젯 > 약관 설정** 메뉴에서 새로운 이용약관을 추가할 수 있어요.

![이용약관 추가](https://static.tosspayments.com/docs/guides/widget/어드민%20사용하기-약관%20설정.png)

새로 추가한 이용약관을 연동하려면 이용약관을 렌더링하는 [`renderAgreement()`](/sdk/v2/js#widgetsrenderagreement)에 위에 설정한 `variantKey`를 넣으세요. 키값을 넣지 않으면 기본 국문 UI가 렌더링됩니다. 결제 UI의 `variantKey`와 독립적으로 사용됩니다.

```plain copyable="false" theme="grey"
const paymentAgreement = paymentWidget.renderAgreement(
  '#agreement',
  { variantKey: 'AGREEMENT' }
)
```
