### 2. 기능 누락 보강 (services.py / services_managers.py / registry.py 대조)
추가·명문화한 항목:
- 서비스 목록: **엑셀 내보내기**(필터 반영, 최대 행 제한), 정렬.
- 등록: 서비스명 중복 거부, 허용 IP는 IPv4 전용·콤마 허용·루프백 자동 제외, 빈 IP = 제한 없음(HMAC만 보호).
- 키: **키 복사 모달**(이미 발급된 키 재확인, 감사 기록, no-store)과 **키 재발급(회전)**을 별도 하위 섹션으로 분리.
- 허용 IP 수정(전체 교체).
- 단건 취소 정책(허용 여부·수수료율 0~100).
- 서비스별 토스 시크릿 키(쓰기 전용, 빈 값 저장 시 기존 유지, set/changed 감사).
- **알림 수신 URL 저장 + 테스트 전송**(http/https 검증, 빈값=끔, 동기 테스트 결과 표시).
- 상태 변경(ACTIVE/INACTIVE) 효과.
- 담당자 관리(추가/대표 변경/해제, 대표 해제 불가).
- 삭제(구독 있으면 거부, 요금제·담당자 User CASCADE).
- 상세 데이터 탭 5종(요금제·구독·단건결제·등록 카드·이벤트) 표로 정리, 이벤트 탭 범위 명시.
