# 다크 모드 — 차트 텍스트 가시성 수정 + 팔레트 고급화 워크로그

- 날짜: 2026-06-24
- 작업자: seungjinhan
- 요청: 다크 모드에서 대시보드 차트 글자가 안 보임(수정) + 다크 모드 전체 색감을 더 고급스럽게.

## 1) 차트 텍스트/격자 가시성 (다크에서 안 보이던 문제)
원인: `app/admin/templates/_charts.html`의 축 라벨·격자선·도넛 트랙이 `rgba(28,28,28,…)`(거의 검정)을
**하드코딩**해 어두운 배경에서 사라짐.

수정 — 하드코딩 색을 **테마 변수 기반 클래스**로 교체(클래스 CSS가 SVG presentation 속성보다 우선):
- 축 라벨/도넛 '전체' 라벨 → `class="chart-axis"`
- 격자선 → `class="chart-grid"`, 도넛 트랙 → `class="chart-track"`
- 도넛 합계 숫자 → `class="chart-total"`(`fill: var(--black)` — 테마 따라 반전)
- `app/static/admin.css`:
  - `.chart-axis { fill: var(--chart-ink, <라이트 기본>) }`, `.chart-grid,.chart-track { stroke: var(--chart-grid, <라이트 기본>) }`, `.chart-total { fill: var(--black) }`
  - 폴백값이 기존 라이트 색이라 **라이트 모드는 그대로**, 다크에서는 `--chart-ink/--chart-grid`(밝은 잉크)로 보인다.

## 2) 다크 팔레트 고급화
`:root[data-theme="dark"]` 토큰을 더 정제된 톤으로 교체:
- 표면: 깊은 쿨 블랙 페이지(`#0d0f14`) 위에 살짝 떠 보이는 카드(`#171a22`)·은은한 보더(`#242936`).
- 텍스트: 눈부시지 않은 오프화이트 계조(`#ECEEF3`/`#B3BBC9`).
- 강조색: 다크에서 블루를 살짝 밝혀(`#6E8BFF`) 또렷하고 고급스럽게(active/링크/배지/차트선 일괄).
- 틴트(딥 네이비)·그림자(깊고 부드럽게) 정비.

## 검증
- Playwright 스크린샷(scratchpad)으로 라이트/다크 육안 검증:
  - 다크 — 축 라벨(1월/5월/12월/전체)·격자·도넛 트랙·합계 숫자 모두 또렷, 깊은 배경+카드 단계감+밝은 블루로 고급스러움.
  - 라이트 — 차트·색감 기존과 동일(회귀 없음).
- `admin.css`/`admin.js` 변경은 토큰·클래스 기반이라 컴포넌트 수정 불필요.

## 비고
