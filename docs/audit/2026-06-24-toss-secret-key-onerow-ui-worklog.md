# 2026-06-24 — 서비스 상세 '토스 시크릿 키' UI 한 줄 레이아웃 변경

## 요청
서비스 상세의 토스 시크릿 키 영역을 한 줄로:
`토스 시크릿 키 (등록여부) [입력박스] 저장버튼 (설명)`

## 변경
- `app/admin/templates/services/detail.html` — 기존 카드(h3 + 현재상태 kv + form + 하단 설명 문단)를 **단일 flex row form**으로 통합.
  - 순서: **라벨**(토스 시크릿 키) → **등록여부 배지**(설정됨/미설정) → **입력박스**(쓰기 전용, `type=password`) → **저장** 버튼 → **설명**(AES 암호화·재표시 안 됨, 작은 muted 텍스트).
  - `flex-wrap`으로 좁은 화면에서 자연 줄바꿈. 입력 placeholder는 상태별("변경 시에만 입력" / "입력 시 저장").
- 동작·엔드포인트(`POST /admin/services/{id}/toss-secret-key`)·보안(쓰기 전용·평문 재표시 금지·빈 값 저장 시 기존 유지)은 **변경 없음**.

## 검증
- Jinja 템플릿 파싱 OK.
- 매뉴얼 `docs/user_manual/02-admin-services.md` §2.8 서술은 동작 기준이라 그대로 정확 → 문서 수정 불필요(레이아웃만 변경).
