### 2. 기능 보강 (코드 대조)
대조한 코드:
- `app/admin/routes/dashboard.py`, `app/services/dashboard.py`
- `app/admin/routes/settlement.py`, `app/services/settlement.py`

보강 내용 (주로 10.4 정산 섹션):
- 정산 화면 **두 가지 모드**(전체 모드 = 서비스별 요약 표 + 소계,
  서비스별 모드 = 결제 건별 목록) 표로 설명.
- 전체 모드 요약 표 컬럼(서비스·건수·구독매출·일반매출·총매출·환불·순매출) 명시.
- 서비스별 모드 건별 목록 컬럼(승인/부분취소/취소 상태 포함) 명시.
- **집계 대상 = DONE + CANCELED만**, 실패·대기 제외, 기준일 = approved_at 참고 콜아웃.
- **기간 기본값 = 당월 1일 ~ 오늘** 자동 설정 명시.
- 서비스 필터(미담당 서비스 지정 시 404 처리), 요금제 필터(구독 결제만 남김) 설명.
- **엑셀 다운로드** 두 형식(전체 모드 = 서비스별 합계 / 서비스별 모드 = 건별 상세) 표.
