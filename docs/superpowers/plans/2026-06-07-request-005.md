# 요청 005 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 첫결제 금액을 정가 기준으로 변경(실제 청구 포함), 요금제 리스트 상시할인 컬럼+계산내역 툴팁, 서비스 상세 UX 개편(상단 액션 버튼·키 복사 모달·관리자 개요 통합·IP 옥텟 입력), 모든 admin 리스트와 서비스 상세 요금제 액션의 htmx 부분 갱신.

**Architecture:** 금액 정책은 `billing_math.plan_first_amount` 한 곳 변경으로 표시+실제 청구에 일관 적용. API 키는 인증용 해시를 유지한 채 표시용 암호화 컬럼을 추가(Alembic). htmx 부분 갱신은 각 리스트의 "툴바+테이블+페이저"를 partial 템플릿로 분리하고 `HX-Request`/`HX-Target` 헤더로 partial만 렌더 — 기존 PRG 리다이렉트는 XHR이 HX-* 헤더를 유지한 채 따라가므로 액션 라우트는 무변경.

**Tech Stack:** FastAPI, Jinja2, htmx 2.0.4(이미 base.html 로드됨), Alembic, vanilla JS, pytest

**스펙:** `docs/superpowers/specs/2026-06-07-request-005-design.md`

```python
def _fmt_won(v: int) -> str:
    return f"{v:,}원"


def recurring_amount_breakdown(plan) -> str:
    """정기 결제액 계산 내역 — 리스트 툴팁 표시용."""
    t = plan.recurring_discount_type
    if t == FirstPaymentType.DISCOUNT_AMOUNT:
        return (f"정가 {_fmt_won(plan.price)} − 상시 할인 "
                f"{_fmt_won(plan.recurring_discount_value or 0)} = "
                f"{_fmt_won(plan_recurring_amount(plan))}")
    if t == FirstPaymentType.DISCOUNT_PERCENT:
        return (f"정가 {_fmt_won(plan.price)} − 상시 할인 "
                f"{plan.recurring_discount_value}% = "
                f"{_fmt_won(plan_recurring_amount(plan))}")
    return f"정가 {_fmt_won(plan.price)}"


def first_amount_breakdown(plan) -> str:
    """첫 결제액 계산 내역 — 정가 기준 첫구독 할인(요청 005)."""
    t = plan.first_payment_type
    if t == FirstPaymentType.FREE:
        return f"첫 회차 무료 = {_fmt_won(0)}"
    if t == FirstPaymentType.DISCOUNT_AMOUNT:
        return (f"정가 {_fmt_won(plan.price)} − 첫구독 할인 "
                f"{_fmt_won(plan.first_payment_value or 0)} = "
                f"{_fmt_won(plan_first_amount(plan))}")
    if t == FirstPaymentType.DISCOUNT_PERCENT:
        return (f"정가 {_fmt_won(plan.price)} − 첫구독 할인 "
                f"{plan.first_payment_value}% = "
                f"{_fmt_won(plan_first_amount(plan))}")
    return f"정가 {_fmt_won(plan.price)}"
```

```js
  function ipRowHtml() {
    var oct = '<input type="text" inputmode="numeric" maxlength="3" class="ip-oct">';
    return '<div class="ip-row" data-ip-row>' + oct + oct + oct + oct +
      '<button type="button" class="btn-text" data-ip-del style="color:#E5396E">삭제</button></div>';
  }

  document.addEventListener("click", function (e) {
    var add = e.target.closest("[data-ip-add]");
    if (add) {
      var rows = add.closest("form").querySelector("[data-ip-rows]");
      rows.insertAdjacentHTML("beforeend", ipRowHtml());
      return;
    }
    var del = e.target.closest("[data-ip-del]");
    if (del) del.closest("[data-ip-row]").remove();
  });

  document.addEventListener("input", function (e) {
    if (!e.target.classList || !e.target.classList.contains("ip-oct")) return;
    var el = e.target;
    el.value = el.value.replace(/\D/g, "").slice(0, 3);
    if (el.value.length === 3) {
      var next = el.nextElementSibling;
      if (next && next.classList.contains("ip-oct")) next.focus();
    }
  });

  document.addEventListener("submit", function (e) {
    var form = e.target;
    if (!form.hasAttribute("data-ip-form")) return;
    var lines = [];
    var bad = false;
    form.querySelectorAll("[data-ip-row]").forEach(function (row) {
      var octs = Array.prototype.map.call(
        row.querySelectorAll(".ip-oct"), function (i) { return i.value.trim(); });
      var filled = octs.filter(Boolean).length;
      row.style.outline = "";
      if (filled === 0) return;  // 빈 행 무시
      var valid = filled === 4 && octs.every(function (v) { return +v >= 0 && +v <= 255; });
      if (!valid) { bad = true; row.style.outline = "1px solid var(--status-error, #E5396E)"; return; }
      lines.push(octs.join("."));
    });
    if (bad) { e.preventDefault(); showToast("IP 형식을 확인하세요 (각 칸 0~255)", "error"); return; }
    form.querySelector('input[name="allowed_ips"]').value = lines.join("\n");
  }, true);  // capture — data-confirm/htmx 핸들러보다 먼저 합성
```

`toolbar` — 시그니처 `{%- macro toolbar(action, pp, placeholder='검색', extra_selects=None, target=None) -%}`, form 태그를:

```html
<form method="get" action="{{ action }}" class="toolbar"{% if target %} hx-get="{{ action }}" hx-target="#{{ target }}" hx-swap="outerHTML" hx-push-url="true"{% endif %}>
```

select의 `onchange="this.form.submit()"`을 `onchange="this.form.requestSubmit()"`로 변경 (form.submit()은 htmx를 우회함 — requestSubmit은 submit 이벤트를 발생시켜 htmx가 가로챈다. 비-htmx 페이지에서도 동일 동작).

초기화 링크를:

```html
{%- if pp.q or pp.filters -%}<a class="btn-text" href="{{ action }}"{% if target %} hx-get="{{ action }}" hx-target="#{{ target }}" hx-swap="outerHTML" hx-push-url="true"{% endif %}>초기화</a>{%- endif -%}
```

`sort_th` — 시그니처 `{%- macro sort_th(pp, action, col, label, target=None) -%}`, 앵커를:

```html
    <a href="{{ action }}?{{ base }}{{ sep }}sort={{ col }}&dir={{ pp.toggled_dir(col) }}"
       {% if target %}hx-get="{{ action }}?{{ base }}{{ sep }}sort={{ col }}&dir={{ pp.toggled_dir(col) }}" hx-target="#{{ target }}" hx-swap="outerHTML" hx-push-url="true"{% endif %}
       style="display:inline-flex;align-items:center;gap:4px;color:inherit">
```

`pager` — 시그니처 `{%- macro pager(page, action, pp, target=None) -%}`. 링크가 5종(이전/1/창/마지막/다음)이므로 내부 헬퍼 매크로로 묶는다. pager 매크로 본문 시작에:

```html
{%- macro _plink(action, qs, label, cls, disabled, target) -%}
<a class="{{ cls }} {{ 'disabled' if disabled }}" href="{{ action }}?{{ qs }}"
   {% if target and not disabled %}hx-get="{{ action }}?{{ qs }}" hx-target="#{{ target }}" hx-swap="outerHTML" hx-push-url="true"{% endif %}
   {{ 'style=pointer-events:none;opacity:.4' if disabled }}>{{ label }}</a>
{%- endmacro -%}
```

를 두고 기존 `<a …>이전</a>`, 숫자, `1`, 마지막, `다음` 링크를 `_plink` 호출로 교체:

```html
    {{ _plink(action, base ~ sep ~ 'page=' ~ (page.page - 1), '이전', 'btn btn-sm btn-ghost', not page.has_prev, target) }}
    {%- if page.window()[0] > 1 -%}
      {{ _plink(action, base ~ sep ~ 'page=1', '1', 'btn btn-sm btn-ghost', false, target) }}
      {%- if page.window()[0] > 2 -%}<span class="muted" style="padding:0 4px">…</span>{%- endif -%}
    {%- endif -%}
    {%- for n in page.window() -%}
      {{ _plink(action, base ~ sep ~ 'page=' ~ n, n, 'btn btn-sm ' ~ ('btn-primary' if n == page.page else 'btn-ghost'), false, target) }}
    {%- endfor -%}
    {%- if page.window()[-1] < page.pages -%}
      {%- if page.window()[-1] < page.pages - 1 -%}<span class="muted" style="padding:0 4px">…</span>{%- endif -%}
      {{ _plink(action, base ~ sep ~ 'page=' ~ page.pages, page.pages, 'btn btn-sm btn-ghost', false, target) }}
    {%- endif -%}
    {{ _plink(action, base ~ sep ~ 'page=' ~ (page.page + 1), '다음', 'btn btn-sm btn-ghost', not page.has_next, target) }}
```

(Jinja에서 macro 안 macro 정의가 동작하지 않으면 `_plink`를 파일 레벨 매크로로 둔다.)

- [ ] **Step 4: admin.js — data-confirm ↔ htmx 연동**

(a) `openConfirm`/ok 핸들러에 콜백 지원 — `var overlay, pendingForm;`을 `var overlay, pendingForm, pendingOk;`로, `openConfirm(form)`을 `openConfirm(form, onOk)`로 바꾸고 본문 첫 줄에 `pendingOk = onOk || null;` 추가. `closeConfirm`에 `pendingOk = null;` 추가. `[data-modal-ok]` 분기를:

```js
    } else if (e.target.closest("[data-modal-ok]")) {
      var form = pendingForm;
      var ok = pendingOk;
      closeConfirm();
      if (ok) { ok(); }                              // htmx 요청 재개
      else if (form) { form.__confirmed = true; form.submit(); }
```

(b) 기존 submit 인터셉터는 htmx 폼을 건너뛴다:

```js
  document.addEventListener("submit", function (e) {
    var form = e.target;
    if (form.hasAttribute("hx-post")) return;  // htmx 폼은 htmx:confirm에서 처리
    if (form.hasAttribute("data-confirm") && !form.__confirmed) {
      e.preventDefault();
      openConfirm(form);
    }
  });
```

(c) htmx 요청용 confirm 후킹 — IIFE 안에 추가:

```js
  // --- htmx 요청의 data-confirm 모달 (요청 005) ---
  document.body.addEventListener("htmx:confirm", function (e) {
    var form = e.detail.elt.closest && e.detail.elt.closest("form[data-confirm]");
    if (!form) return;                 // data-confirm 없으면 그대로 진행
    e.preventDefault();                // 요청 보류
    openConfirm(form, function () { e.detail.issueRequest(true); });
  });
```
