1. 2가지 종류의 메뉴얼
   1. 사용자 메뉴얼
      1. 구독/결제 서버를 처음 사용하는 사람이 전반적인 서비스를 이해하고 이용할수 있는 메뉴얼
      2. 메뉴는
         1. 그림으로 전체 프로세스를 설명
         2. 관리자 콘솔에 대한 설명
            1. 카드관리
            2. 구독관리
            3. 요금제관리
            4. 일반결제, 환불
            5. 계정관리
            6. 전체설졍
            7. 감사로그
         3. 대시보드설명
            1. 구독
            2. 결제
            3. 정산
            4. 요금제
   2. 개발자메뉴얼
      1. 설치.설정.배포(docker 설정 및 배포까지)
      2. 서비스에서 사용할 기능
         1. 서비스에서 사용할 API 설명
         2. 전체 기능별로 상세설명 및 유지보수를 위한 코드설명(호출시작부터 반환까지 코드설명 다 포함)
      3. 어드민페이지 화면별 설명
