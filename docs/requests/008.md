1. 구동
   1. 구독을 위해서는 결제정보를 저장하고 해당일이 되면 자동결제를 진행해야 한다.
   2. 자동결제를 위한 알고리즘은 추가 되어 있는것인가?
   3. 구독정보에 만료일 앞에 다음결제예정일(시간)을 추가 - 만료시간 12시간전
   4. Expired 되면 결제정보는 다시 사용할수 없게 정보를 hiding처리 해야 한다.(보안을 위해서)
2. 대시보드
   1. 첫화면은 이번달상황(월을 선택가능)
      1. 현재 전체 구독개수 (서비스별 구독)
      2. 신규구독수
      3. 이번달 매출금액
      4. 이번달 미결제 수
      5. 구독 취소개수(사용자취소, 결제만료 구분)
      6. 구독상태
   2. 차트
      1. 최근 12개월 매출, 신규구독, 전체구독수, 구독취소개수
   3. 토탈정보
      1. 누적 구독개수, 매출(서비스별)
3. 위의 각 항목들은 누르면 상세내역을 보는 화면으로 이동해야 함.
4. 오른쪽에 최근결제내역, 현재미수구독
5. 그외 대시보드에 들어갈 필요가 있는 정보를 추가 해줘
