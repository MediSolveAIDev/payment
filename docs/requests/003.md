1. 이메일전송
   1. .env에 구글 계정정보가 있다.
   2. 구글 메일 전송처리 해야 한다

2. 계정
   1. 정보추가
      1. 전화번호(필수아님)

   2. 정보 수정기능
      1. 이메일수정시 반드시 중복체크

   3. 비밀번호 재설정 메일
      1. 비밀번호 재설정 메일을 보냄
      2. 사용자가 메일에서 링크를 클릭하면 재설정 페이지 열리고 입력하면 변경처리

3. 상태
   1. 관리자가 사용자 생성을 하면 pending 이고 사용자에게 재설정 메일을 보내서 패스워드 입력하면 활성화 완료
   2. 관리자가 사용자 비활성화 및 삭제처리 가능
      1. 각 버튼을 누를때는 경고 메세지를 보여줘야 함

4. 요금제
   1. 첫구독 할인 외에
   2. 요금 할인 정보를 넣을수 있어야 한다.
      1. 할인율 and 할인금액
   3. 리스트와 상세에 할인했을경우 실제 결제할 금액 정보도 나와야 한다.

5. 구독리스트 - 필터에 서비스를 선택가능(전체, 서비스명), 만료일 기준으로 정렬가능해야 함
