1. 서비스
   1. 요금제
      1. 상세보기 에서 요금제 관리를 해야 함
      2. 요금제는 수정/비활성화/삭제
      3. 할인률 입력기능
      4. 반복회차 정보 필요
      5. 구독시작일
      6. 만료일
   2. 관리자 할당기능
   3. 키복사기능필요
2. 계정
   1. 계정추가 기능
   2. 계정은 2타입 - 슈퍼관리자, 서비스관리자
      1. 슈퍼관리자는 전체를 관리
      2. 서비스관리자는 본인에게 할당된 서비스만 관리
         1. 1명의 관리자는 여러개의 서비스 관리 가능
3. 감시로그
   1. 관리자가 이해할수 있는 데이터가 나와야 함
4. 전체
   1. 모든 리스트는 페이징 처리(15개 기본)
   2. 검색, 필터, 정렬기능 필요
