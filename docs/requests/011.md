1. 구독
   1. 리스트에 전체서비스 옆에 요금제를 추가 해줘(요금제는 서비스에 해당하는 요금제가 나와야 함)
2. 결제
   1. 결제이력에 전체서비스 옆에 요금제추가(요금제는 서비스에 해당하는 요금제가 나와야함)
3. 정산
   1. 정산 검색조검에 전체서비스 옆에 요금제추가(요금제는 서비스에 해당하는 요금제가 나와야함)
   2. 전체정산금액 아래에 구독 정산금액(횟수), 일반결제정산금액(횟수)
   3. 리스트에 일반결제에 대해서도 상세보기 버튼넣고 누르면 상세정보 보여줘
4. 대시보드
   1. 구독에 구독상태에 만료데이터 추가
   2. 상태나 종류를 나타내는 enum값들은 색을 다르게 변경해줘
