1. 요금제
   1. 첫결제 금액은 상시할인과 상관없이 원래 요금으로만 적용
   2. 요금제 리스트에 상시할인 항목을 만들어서 넣어줘
      1. 정기결제 금액에 상시할인 비율은 제거
   3. 첫결제액, 정기결재액에 마우스를 올리면 계산 내역을 보여줘

2. 서비스 상세
   1. 허용IP는 한줄씩 넣어야 한다. 그 한줄도 4개의 box가 있고 입력(숫자만 입력)
   2. 비활성화, 키재발급, 삭제 버튼은 상단에 서비스 이름옆에 배열
   3. 키 복사 기능 추가(버튼누르면 모달이 나오고 그 모달에서 키를 복사 버튼)
   4. 관리자 할당은 개요에 담당자 추가버튼을 두고 담당자 이메일 옆에 삭제,수정버튼 추가
   5. 요금제의 비활성화, 삭제 버튼 클릭시, 화면 재로딩이 이난 해당 부분만 화면 영역 변화
   6. 각 리스트의 정렬 할때 화면 전체 재로딩이 아닌 리스트만 변경되어야 함
