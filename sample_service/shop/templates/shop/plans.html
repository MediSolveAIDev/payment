호출하는 API

GET /api/v1/plans — 활성 서비스의 요금제 목록 (HMAC 3중 인증)

구현 위치

  • API 호출: shop/payment_client.py → get_plans()
  • 화면 로직: shop/views.py → plans_view()

핵심 규칙

  • 응답의 amount는 상시 할인 적용 후 실제 정기 결제액 — 서버가 계산해 내려준다
  • 첫 결제액 표시는 이 샘플이 서버 정책(first_payment_type/value)을 미러해 계산 — 실제 청구액 산정은 항상 서버가 한다
  • "무료 체험" 버튼은 같은 카드 등록 흐름에 ?trial=1만 추가 — 체험은 결제 없이 시작되고 만료 시 자동 결제된다
